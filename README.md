# ⚖️ Defensa Legal — Landing

**Estudio jurídico penal especializado en defensa estratégica de personas imputadas por delitos en Chile.**

Landing page institucional optimizada para SEO y conversión, construida con **Vite**, **React**, **TypeScript** y **Tailwind CSS**, desplegada en **Vercel**.

---

## 🚀 Características Principales

✅ Vite para desarrollo ultrarrápido  
✅ React 18 con enrutamiento suave (scroll behavior)  
✅ TypeScript para robustez y escalabilidad  
✅ Tailwind CSS para diseño elegante y responsive  
✅ Optimización SEO legal y local  
✅ Deploy en Vercel con performance excelente  
✅ Animaciones y degradados responsivos  
✅ Diseño accesible y mobile-first  
✅ Logo responsivo y paleta personalizada (rojo vino + azul petróleo)

---

## 🛠️ Instalación y Desarrollo

```bash
# Clona el repositorio
git clone <repository-url>
cd defensa-penal-protegida

# Instala las dependencias
npm install

# Ejecuta el servidor de desarrollo
npm run dev
```
